# Assignment 4: Heap Data Structures – Implementation, Analysis, and Applications

# How to Run

## Prerequisites

- Install Python 3.x on your computer.
- Make sure all Python files are in the same project folder.

## Run the HeapSort Program

Open a terminal (or Command Prompt), navigate to the project folder, and run:

```bash
python heapsort.py
```

## Run the Priority Queue Program

```bash
python priority_queue.py
```

## Run the Benchmark Program

```bash
python benchmark.py
```

## Notes

- `heapsort.py` demonstrates the HeapSort algorithm.
- `priority_queue.py` demonstrates the Priority Queue operations using a Binary Max Heap.
- `benchmark.py` compares the execution time of HeapSort, QuickSort, and MergeSort.
